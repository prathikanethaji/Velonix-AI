## 📌 Overview

Velonix AI is an intelligent bike demand forecasting platform built using **Machine Learning** and **Flask**.

The application helps predict bike rental demand based on weather and time-related conditions. It also provides an analytics dashboard and supports bulk prediction using CSV uploads.

---

# ✨ Features

### 🚲 Single Prediction
Predict bike rental demand using user inputs.

### 📂 Batch Prediction
Upload a CSV file and predict demand for multiple records at once.

### 📊 Analytics Dashboard
Visualize demand trends using interactive charts.

### 🤖 AI Recommendation Engine
Provides recommendations based on predicted demand levels.

### 📥 Download Results
Download prediction results as a CSV file.

### 📄 Sample CSV
Download a sample CSV template for batch predictions.
